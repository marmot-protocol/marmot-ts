# Code Examples

These practical examples demonstrate common use cases when building with `marmot-ts`.

## 1. Creating a Group

When creating a group, you must define the base group info (`name`, `description`, etc.) and the list of initial admin pubkeys. The library automatically adds the creator to the group.

```typescript
import { bytesToHex } from "@noble/hashes/utils";

// Make sure you have an initialized MarmotClient instance
const group = await client.createGroup("Engineering Sync", {
  description: "Secure comms for the backend team",
  relays: ["wss://relay.nostr.info"],
  // Optional: add additional admins (the creator is always included automatically)
  adminPubkeys: ["<other-admin-pubkey-hex>"],
  // Optional: override MLS ciphersuite
  ciphersuite: "MLS_128_DHKEMX25519_AES128GCM_SHA256_Ed25519",
});

// The group ID is a hash generated by MLS (Uint8Array)
console.log(`Created group with ID: ${bytesToHex(group.id)}`);
```

## 2. Inviting a Member

To invite a new member, the group admin must first retrieve one of their public Key Package events (Kind `443`), and then pass it to the group to create an encrypted Welcome message. This message is then sent via a NIP-59 Gift Wrap.

```typescript
// 1. Fetch the user's key package from the network
const memberPubkey = "abc123def456...";
const keyPackageEvent = await client.network
  .request(
    ["wss://relay.example.com"],
    [{ kinds: [443], authors: [memberPubkey], limit: 1 }],
  )
  .then((events) => events[0]);

// 2. Propose, Commit, and send the Invite in one action
if (keyPackageEvent) {
  // Returns a mapping of Nostr relay URLs to their PublishResponses
  const publishResults = await group.inviteByKeyPackageEvent(keyPackageEvent);
  console.log(
    "Successfully invited user. Relays responded with:",
    publishResults,
  );
}
```

## 3. Joining a Group

If someone invited you, their Welcome message will arrive in your NIP-65 inbox as a Gift Wrap (`1059`). After decrypting the Gift Wrap to extract the Rumor (`444`), you instantiate the group.

```typescript
// Assuming you've already decrypted a NIP-59 gift wrap
// `inviteRumor` is the unsealed Kind 444 event.
const inviteRumor = {
  /* ... */
};

// Find the associated key package ID in the event tags
const keyPackageEventId = inviteRumor.tags.find((t) => t[0] === "e")?.[1];

const group = await client.joinGroupFromWelcome({
  welcomeRumor: inviteRumor,
  keyPackageEventId,
});

console.log(`Joined group: ${bytesToHex(group.id)}`);
```

## 4. Sending a Message

Messages sent within a group are packaged as `applicationMessage` payloads. You construct a generic Nostr event rumor (e.g. Kind `9` Chat Message) and pass it to the group. The group encrypts the event and publishes it as a Kind `445`.

```typescript
import { getEventHash } from "applesauce-core/helpers";

const myMessageContent = "Hello everyone, the server is deployed!";

// Create a generic Nostr event rumor
const rumor = {
  kind: 9,
  pubkey: await client.signer.getPublicKey(),
  created_at: Math.floor(Date.now() / 1000),
  content: myMessageContent,
  tags: [],
  id: "",
};
rumor.id = getEventHash(rumor);

// Send the message through the MLS group
await group.sendApplicationRumor(rumor);
```

## 5. Receiving Messages

To receive messages, you subscribe to Kind `445` events on the group's specific relays, and feed the stream into the `group.ingest()` method.

```typescript
import { deserializeApplicationRumor } from "@internet-privacy/marmots";

// 1. Subscribe to the group's network relay(s)
const subscription = client.network.subscription(group.groupInfo.relays, [
  { kinds: [445], "#h": [bytesToHex(group.id)] },
]);

// 2. Ingest events as they arrive
subscription.subscribe({
  next: async (event) => {
    const results = group.ingest([event]);

    for await (const result of results) {
      if (result.kind === "applicationMessage") {
        // Decode the MLS binary payload back into a Nostr Rumor
        const message = deserializeApplicationRumor(result.message);
        console.log(`New message from ${message.pubkey}: ${message.content}`);
      }
    }
  },
});
```
